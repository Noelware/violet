**Violet** by Noelware, LLC. is a fresh, new take on an extended C++ standard library. It provides implementations that tie into C++ features and types, but with eyes from a prior Rust developer who loves Rust's abstractions over C++'s.

There've been many issues with the implementations of C++'s standard library and **Violet** is one implementation over the variety available over the years. We know that this isn't unique at all, but when developing our software like [Eous](https://eous.noelware.cloud), this would be essential to build out. Some abstractions that were built from **Eous** also migrate into Violet if we see fit.

> [!IMPORTANT]
> **Violet** and its frameworks are ***VERY EXPERIMENTAL!!!***
>
> While we do work to improve issues that we haven't caught or from CI, we are not claiming that Violet and subframeworks are ready for production yet.
